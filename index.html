Envoi WhatsApp — Formulaire PWA

Ajoutez sur l’écran d’accueil pour un accès instantané, même hors ligne.

Aperçu
Importez une image (JPG/PNG, max ~3 Mo). Elle n’est pas envoyée à WhatsApp, mais peut être sauvegardée localement.
Astuce: l’image est stockée localement (cache PWA) pour réutilisation rapide.
Veuillez saisir un nom valide.
Le numéro ci‑dessous doit être sans zéro initial ni espaces (ex: 0701… devient 701…).
Entrez un numéro valide (6 à 15 chiffres, sans espaces).
Les champs requis: Nom, Numéro. Le message est pré-rempli avec le nom.
